# DevOps

## Infracture as Code
サーバやネットワークなどのインフラ構成をコード化し、ソフトウェア開発の考え方やスタイルをインフラ運用に取り入れること

## ウォーターフォールからアジャイルへ
変化の激しい開発現場に対応するため、短いスパンで繰り返しリリースをしていくアジャイル開発が注目されてきている
頻繁なリリースに対応するためにさまざまな自動化ツールが使われるようになった
